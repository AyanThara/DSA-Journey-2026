# Arrays in C++

This folder contains basic **array implementations in C++** for learning **DSA fundamentals**.

## Concepts Covered
- Array declaration and initialization  
- Traversing arrays  
- Passing arrays to functions  
- Finding max/min elements  
- Linear search  
- Reversing an array  

## Notes
- Arrays are passed to functions by reference.
- STL is not used here to focus on core logic.
